Welcome to the user study for the **BabaIsBot** Abductive Reasoning experiment.

This study aims to evaluate whether an agent is truly engaging in abductive reasoning (inferring rules from observation) or simply relying on its prior knowledge of the game *Baba Is You*. To test this, we present game states where the **Semantics** (meaning) and **Syntax** (grammar) of the rules have been transformed.

## ðŸ” Prompt Conditions Key

* **Base:** The original game state text.
* **Rephrase:** Semantics and syntax preserved, but words are synonyms (e.g., `BABA IS YOU` -> `PLAYER CONTROLS BABA`).
* **Break Syntax:** Semantics preserved, but rule structure is broken (e.g., `YOU RELATES BABA`).
* **Break Semantics:** Syntax preserved, but words are swapped using the dictionary above (e.g., `CIRCLE IS GREEN`).
* **Break Semantics & Syntax:** Both dictionary swaps and broken rule structure are applied (e.g., `GREEN RELATES CIRCLE`).
